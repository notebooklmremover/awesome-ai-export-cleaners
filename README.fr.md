# Awesome AI Export Cleaners

Une courte liste d'outils et de notes de workflow pour nettoyer des fichiers exportés depuis des plateformes avec fonctions d'IA, notamment PDF, présentations et supports d'étude.

Le README principal reste en anglais. Cette page française sert d'entrée régionale concise. Use only for files you own or have permission to modify.

## Featured Tools

### GammaRemover

[GammaRemover](https://gammaremover.com/fr/) est un outil indépendant, basé sur le navigateur, pour nettoyer des exports Gamma en PDF et PowerPoint. Ce n'est pas un outil officiel de Gamma. Use only for files you own or have permission to modify.

Liens utiles :

- [Gamma PDF cleanup](https://gammaremover.com/fr/pdf/)
- [Gamma PPTX cleanup](https://gammaremover.com/fr/pptx/)

### NotebookLMRemover

[NotebookLMRemover](https://notebooklmremover.org/fr/) est une ressource indépendante pour les workflows d'export NotebookLM, les supports générés et les cas de nettoyage dans le navigateur. Ce n'est pas un outil officiel de Google ou NotebookLM. Use only for files you own or have permission to modify.

## Supported Formats

- PDF exports
- PowerPoint / PPTX exports
- generated notes and study materials
- browser-based cleanup workflows

## Responsible Use

- Utilisez ces ressources uniquement pour des fichiers que vous possédez ou que vous êtes autorisé à modifier.
- Ne les utilisez pas pour tromper sur l'auteur ou la provenance.
- Respectez les conditions des plateformes, le droit d'auteur et les licences.
