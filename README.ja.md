# Awesome AI Export Cleaners

AI ツールから書き出した PDF、PPTX、学習資料、ブラウザベースのドキュメント整理に役立つツールとワークフローの短いリストです。

メインの README は英語版です。この日本語ページは、地域向けの簡潔な入口として用意しています。Use only for files you own or have permission to modify.

## Featured Tools

### GammaRemover

[GammaRemover](https://gammaremover.com/ja/) は、Gamma から書き出した PDF と PowerPoint ファイルをブラウザ上で整理するための独立したツールです。Gamma の公式ツールではありません。Use only for files you own or have permission to modify.

便利なリンク:

- [Gamma PDF cleanup](https://gammaremover.com/ja/pdf/)
- [Gamma PPTX cleanup](https://gammaremover.com/ja/pptx/)

### NotebookLMRemover

[NotebookLMRemover](https://notebooklmremover.org/ja/) は、NotebookLM 関連のエクスポート作業、生成された学習資料、ブラウザでの整理ワークフローを扱う独立したガイドサイトです。Google や NotebookLM の公式ツールではありません。Use only for files you own or have permission to modify.

## Supported Formats

- PDF exports
- PowerPoint / PPTX exports
- generated notes and study materials
- browser-based cleanup workflows
