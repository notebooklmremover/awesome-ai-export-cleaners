# Contributing

Thanks for helping improve Awesome AI Export Cleaners. This repository is meant to stay useful, practical, and trustworthy as a resource directory.

## What Can Be Submitted

You can submit:

- browser-based cleanup tools for AI-assisted exports;
- workflow guides for PDF, PPTX, generated documents, or study materials;
- notes that explain export differences or review steps;
- corrections to supported formats, platform scope, or responsible-use language.

Please avoid submissions that are only keyword pages, thin affiliate pages, or unrelated product listings.

## Required Fields for Tool Submissions

For each tool, include:

- name;
- official URL;
- platform or workflow scope;
- supported formats;
- whether it is browser-based;
- short description;
- responsible-use note.

If a tool is not official to the platform it supports, say that clearly.

## Responsible-Use Requirements

Every submission must make clear that users should only modify files they own or have permission to edit.

Submissions must not encourage:

- misrepresenting authorship;
- violating platform terms, copyright, or licensing rules;
- distributing malware;
- pretending to be an official platform tool;
- spammy affiliate listings or misleading redirect pages;
- tools intended to strip ownership or licensing information from third-party material.

## Pull Request Checklist

Before opening a pull request, check that:

- the resource is real and accessible;
- the description is factual and not keyword-stuffed;
- links are not repeated unnecessarily;
- the submission includes responsible-use context;
- the resource fits the repository structure.
