# Responsible Use

This repository lists tools, guides, and workflow notes for legitimate AI export cleanup. It is intended for people cleaning their own files, preparing internal review materials, or organizing generated documents and study resources they are allowed to edit.

## General Responsible-Use Policy

Use listed resources only for files you own or have permission to modify. Cleanup should support review, organization, accessibility, and format preparation. It should not be used to confuse authorship, hide source context, or ignore platform requirements.

## Ownership and Permission

Before using any cleanup workflow, confirm that you have the right to edit the file. This may mean you created it, your team owns it, or the file license allows modification.

If you are unsure, keep the original file unchanged and ask the file owner before editing.

## Attribution

Do not use cleanup tools to misrepresent authorship. Keep attribution, source notes, and required acknowledgements when they apply.

For generated or AI-assisted materials, review the output carefully and preserve enough context for readers to understand where the material came from.

## Platform Terms

Respect platform terms, copyright, licensing rules, and export settings. When you need a fully supported platform-approved watermark-free export, use official paid export options.
