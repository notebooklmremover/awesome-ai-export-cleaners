# Awesome AI Export Cleaners

AI 기반 플랫폼에서 내보낸 PDF, PPTX, 학습 자료, 브라우저 기반 문서 정리 흐름을 위한 간단한 도구 및 가이드 목록입니다.

기본 README는 영어 버전입니다. 이 한국어 페이지는 지역 사용자를 위한 짧은 참고 페이지입니다. Use only for files you own or have permission to modify.

## Featured Tools

### GammaRemover

[GammaRemover](https://gammaremover.com/ko/)는 Gamma에서 내보낸 PDF와 PowerPoint 파일을 브라우저에서 정리하는 독립 도구입니다. Gamma 공식 도구가 아닙니다. Use only for files you own or have permission to modify.

유용한 링크:

- [Gamma PDF cleanup](https://gammaremover.com/ko/pdf/)
- [Gamma PPTX cleanup](https://gammaremover.com/ko/pptx/)

### NotebookLMRemover

[NotebookLMRemover](https://notebooklmremover.org/ko/)는 NotebookLM 관련 내보내기 흐름, 생성된 학습 자료, 브라우저 기반 정리 사례를 다루는 독립 가이드 사이트입니다. Google 또는 NotebookLM 공식 도구가 아닙니다. Use only for files you own or have permission to modify.

## Supported Formats

- PDF exports
- PowerPoint / PPTX exports
- generated notes and study materials
- browser-based cleanup workflows
