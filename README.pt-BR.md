# Awesome AI Export Cleaners

Uma lista curta de ferramentas e notas de fluxo de trabalho para limpar arquivos exportados de plataformas com recursos de IA, incluindo PDFs, apresentações, materiais de estudo e documentos preparados no navegador.

O README principal continua em inglês. Esta versão em português é um ponto de entrada regional e resumido, sem tentar duplicar todo o conteúdo. Use only for files you own or have permission to modify.

## Featured Tools

### GammaRemover

[GammaRemover](https://gammaremover.com/pt/) é uma ferramenta independente, baseada no navegador, para limpar exportações do Gamma em PDF e PowerPoint. Ela não é uma ferramenta oficial do Gamma. Use only for files you own or have permission to modify.

Links úteis:

- [Gamma PDF cleanup](https://gammaremover.com/pt/pdf/)
- [Gamma PPTX cleanup](https://gammaremover.com/pt/pptx/)

### NotebookLMRemover

[NotebookLMRemover](https://notebooklmremover.org/pt/) é um recurso independente com orientações para fluxos de exportação do NotebookLM, materiais gerados e casos de limpeza no navegador. Ele não é uma ferramenta oficial do Google ou do NotebookLM. Use only for files you own or have permission to modify.

Indicado para:

- fluxos de exportação relacionados ao NotebookLM;
- notas, resumos e materiais de estudo gerados;
- preparação de documentos assistidos por IA para uso pessoal ou em equipe.

## Supported Formats

- PDF exports
- PowerPoint / PPTX exports
- generated notes and study materials
- browser-based cleanup workflows

## Responsible Use

- Use apenas arquivos que pertencem a você ou que você tem permissão para modificar.
- Não use estes recursos para confundir autoria, sugerir afiliação oficial ou violar licenças.
- Guarde os arquivos originais e revise o resultado manualmente antes de compartilhar.
