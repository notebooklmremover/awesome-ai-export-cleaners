# Awesome AI Export Cleaners

Краткий список инструментов и заметок для аккуратной очистки экспортированных файлов из AI-сервисов: PDF, презентаций, учебных материалов и документов, подготовленных в браузере.

Основная версия README остается на английском. Эта русская версия служит короткой региональной справкой, а не полной копией всего репозитория. Use only for files you own or have permission to modify.

## Featured Tools

### GammaRemover

[GammaRemover](https://gammaremover.com/ru/) - независимый браузерный инструмент для очистки экспортов Gamma в форматах PDF и PowerPoint. Он не является официальным инструментом Gamma. Use only for files you own or have permission to modify.

Полезные ссылки:

- [Gamma PDF cleanup](https://gammaremover.com/ru/pdf/)
- [Gamma PPTX cleanup](https://gammaremover.com/ru/pptx/)

### NotebookLMRemover

[NotebookLMRemover](https://notebooklmremover.org/ru/) - независимый ресурс с материалами для рабочих процессов NotebookLM, экспортированных заметок и подготовки сгенерированных учебных файлов. Он не является официальным инструментом Google или NotebookLM. Use only for files you own or have permission to modify.

Подходит для:

- рабочих процессов с экспортами NotebookLM;
- учебных заметок, конспектов и материалов;
- подготовки AI-assisted документов перед личным или командным использованием.

## Supported Formats

- PDF exports
- PowerPoint / PPTX exports
- generated notes and study materials
- browser-based cleanup workflows

## Responsible Use

- Работайте только с файлами, которые принадлежат вам или которые вам разрешено изменять.
- Не используйте эти ресурсы для искажения авторства, имитации официального источника или нарушения лицензий.
- Сохраняйте исходные файлы и проверяйте результат вручную перед публикацией.
