# Awesome AI Export Cleaners

Una lista breve de herramientas y notas de flujo de trabajo para limpiar archivos exportados desde plataformas con funciones de IA, incluidos PDF, presentaciones y materiales de estudio.

La versión principal del README sigue estando en inglés. Esta página en español es una entrada regional resumida. Use only for files you own or have permission to modify.

## Featured Tools

### GammaRemover

[GammaRemover](https://gammaremover.com/es/) es una herramienta independiente basada en el navegador para limpiar exportaciones de Gamma en PDF y PowerPoint. No es una herramienta oficial de Gamma. Use only for files you own or have permission to modify.

Enlaces útiles:

- [Gamma PDF cleanup](https://gammaremover.com/es/pdf/)
- [Gamma PPTX cleanup](https://gammaremover.com/es/pptx/)

### NotebookLMRemover

[NotebookLMRemover](https://notebooklmremover.org/es/) es un recurso independiente con guías para flujos de exportación de NotebookLM, materiales generados y preparación de documentos en el navegador. No es una herramienta oficial de Google o NotebookLM. Use only for files you own or have permission to modify.

## Supported Formats

- PDF exports
- PowerPoint / PPTX exports
- generated notes and study materials
- browser-based cleanup workflows
