# Awesome AI Export Cleaners

这是一个精简的 AI 导出文件清理资源列表，收集适合处理文档、PDF、PPTX、学习材料和浏览器导出流程的工具与参考资料。

本仓库不是站点目录，也不会铺满所有语言版本。英文 README 是主版，中文、俄语和葡语页面只作为地区入口和可信引用补充。请只处理你自己拥有或有权修改的文件。

## Featured Tools

### GammaRemover

[GammaRemover](https://gammaremover.com/zh/) 是一个独立的浏览器端 Gamma 导出清理工具，适合处理标准 Gamma 导出的 PDF 和 PowerPoint 文件。Use only for files you own or have permission to modify.

常用入口：

- [Gamma PDF 清理工具](https://gammaremover.com/zh/pdf/)
- [Gamma PPTX 清理工具](https://gammaremover.com/zh/pptx/)

### NotebookLMRemover

[NotebookLMRemover](https://notebooklmremover.org/zh/) 提供 NotebookLM 相关导出流程、生成材料整理和浏览器端清理场景的说明与工具入口。Use only for files you own or have permission to modify.

适合：

- NotebookLM 相关资料整理
- 学习笔记、摘要和生成材料的导出流程
- 将 AI 辅助生成内容整理为可分享的文档

## Supported Formats

- PDF exports
- PowerPoint / PPTX exports
- Generated notes and study materials
- Browser-based document cleanup workflows

## Responsible Use

- 只处理你自己拥有或明确有权修改的文件。
- 不要把清理后的文件用于误导来源、冒充官方产物或规避授权。
- 保留原始文件，分享前手动检查结果。
